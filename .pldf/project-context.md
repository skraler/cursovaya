# Контекст проекта: Визуализатор кратчайших маршрутов

**Последнее обновление**: 21.05.2026 (implement)

## Обзор проекта

Веб-приложение для построения графа на холсте и расчёта кратчайшего маршрута алгоритмом Дейкстры с обязательными промежуточными остановками в заданном порядке. Пользователь визуально создаёт вершины и рёбра, выбирает старт, финиш и промежуточные точки, получает подсвеченный маршрут и суммарную длину пути.
