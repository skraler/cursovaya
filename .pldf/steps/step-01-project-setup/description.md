# Шаг 01: Настройка проекта Vite + Vitest

**Номер**: step-01-project-setup  
**Дата создания**: 21.05.2026  
**Зависимости**: нет

## Описание

Создать рабочий каркас приложения в корне `cursovaya/` (рядом с `.pldf/`): Vite, Vitest, структура папок `src/` по [architecture.md](../../architecture.md). После шага команда `npm run dev` открывает страницу с заголовком и пустым layout (три колонки-заглушки), `npm test` запускает пустой или smoke-тест.

Это фундамент для всех P1/P2 функций; без него нельзя подключать граф, Store и SVG.

## Задачи

- [ ] `npm create vite@latest .` или ручной `package.json` с `vite`, `vitest` — [package.json](../../../package.json)
- [ ] `vite.config.js`, `vitest.config.js` — корень проекта
- [ ] `index.html` — точка входа Vite
- [ ] Создать `src/main.js` — подключение стилей и mount
- [ ] Создать `src/styles/workspace.css` — базовые переменные из прототипа (светлая схема)
- [ ] Создать пустые каталоги: `src/graph/`, `src/algorithms/`, `src/state/`, `src/ui/`
- [ ] `tests/smoke.test.js` — один тест `expect(true).toBe(true)` или импорт main
- [ ] `README.md` в корне — как запустить `npm install`, `npm run dev`, `npm test`
- [ ] Добавить `.gitignore` записи для `node_modules/`, `dist/` (если ещё нет)
