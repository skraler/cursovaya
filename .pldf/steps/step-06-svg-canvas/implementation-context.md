# Контекст реализации: step-06-svg-canvas

**Завершён**: 21.05.2026

## Реализовано

- Полный `canvasRenderer.js`: SVG рёбра/вершины, веса, стрелки directed
- Клики: ADD_VERTEX, edge 2-click + модалка, SET_ROUTE_POINT, DELETE_VERTEX/EDGE
- Перетаскивание вершин (режим move)
- Подсветка маршрута (`route` class по `fullPathEdgeIds`)
- `svgCoords.js` + unit-тест `findVertexAt`

1. Вершины на холсте (4 клика)
2. Рёбра: A–B (4), B–C (7), C–D (3) — ориентированные
3. Старт A, остановки B, C, финиш D → «Построить маршрут» → **14**, зелёные рёбра
