# Прототип UI — Визуализатор кратчайших маршрутов

## Просмотр в браузере

### Вариант 1: Python (рекомендуется)

Из корня проекта `cursovaya`:

```bash
cd .pldf/ui-prototype
python -m http.server 8000
```

Откройте: [http://localhost:8000/index.html](http://localhost:8000/index.html)

### Вариант 2: Cursor Simple Browser

1. Запустите сервер (команда выше).
2. `Ctrl+Shift+P` → **Simple Browser: Show** → введите `http://localhost:8000/index.html`

### Вариант 3: файл напрямую

Откройте `.pldf/ui-prototype/index.html` в браузере (двойной клик). Навигация между страницами работает.

## Файлы прототипа

| Файл | Назначение |
|------|------------|
| `index.html` | Навигация по экранам |
| `workspace.html` | Главный экран: редактор, маршрут, анимация «Пуск» |

## Проверка анимации (P2)

На `workspace.html`:

1. Нажмите **«Построить маршрут»** — рёбра маршрута станут зелёными.
2. Нажмите **«▶ Пуск — анимация»** — рёбра поочерёдно плавно закрашиваются ярко-зелёным.
3. **«Сбросить подсветку»** — сброс визуализации.

## Вне области прототипа

Сохранение, загрузка и экспорт схемы не проектируются (по решению на этапе дизайна).
