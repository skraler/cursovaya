# Визуализатор кратчайших маршрутов

Веб-приложение для построения графа на холсте и расчёта кратчайшего маршрута (алгоритм Дейкстры) с обязательными промежуточными остановками в заданном порядке.

**Репозиторий:** https://github.com/skraler/cursovaya

---

## Инструкция по запуску (для преподавателя)

### 1. Что нужно установить

| Компонент | Версия | Проверка |
|-----------|--------|----------|
| [Node.js](https://nodejs.org/) (LTS) | 20 или новее | `node -v` |
| npm (идёт с Node.js) | — | `npm -v` |

Если команды не найдены — установите Node.js LTS с официального сайта и перезапустите терминал.

### 2. Получение кода

```bash
git clone https://github.com/skraler/cursovaya.git
cd cursovaya
```

Либо скачайте архив репозитория с GitHub (Code → Download ZIP) и распакуйте его.

### 3. Установка зависимостей

В корне проекта (где лежит `package.json`):

```bash
npm install
```

Выполняется один раз; создаётся папка `node_modules/`.

### 4. Запуск приложения

```bash
npm run dev
```

В терминале появится адрес, например:

```text
  ➜  Local:   http://localhost:5173/
```

Откройте этот адрес в браузере (Chrome, Firefox или Edge). Остановка сервера: `Ctrl+C` в терминале.

> Если порт 5173 занят, Vite предложит другой (например `http://localhost:5174/`) — используйте тот, что указан в выводе.

### 5. Проверка тестов (по желанию)

```bash
npm test
```

Ожидается: все тесты пройдены (`Tests … passed`).

### 6. Краткий сценарий проверки в браузере

1. Слева выберите **«Вершина»** и кликните по холсту 3–4 раза — появятся вершины графа.
2. Выберите **«Ребро (2 клика)»**, укажите две вершины; в окне задайте вес и тип ребра (ориентированное / неориентированное).
3. Задайте **Старт**, при необходимости **Остановку**, **Финиш** (клик по вершинам).
4. Справа нажмите **«Построить маршрут»** — подсветится путь и суммарная длина.
5. Кнопка **«▶ Пуск»** — пошаговая анимация обхода рёбер маршрута.

Документация по проекту и план разработки — в каталоге `.pldf/`.

## Установка и запуск (кратко)

```bash
npm install
npm run dev
```

Откройте в браузере адрес, который выведет Vite (обычно http://localhost:5173).

## Сборка production-версии

```bash
npm run build
npm run preview
```

Статические файлы появятся в каталоге `dist/`. Команда `preview` поднимает локальный сервер для просмотра собранной версии.

## Тесты

```bash
npm test
```

## Структура проекта

| Каталог / файл | Назначение |
|----------------|------------|
| `src/graph/` | Модель графа (вершины, рёбра) |
| `src/algorithms/` | Дейкстра, построение маршрута |
| `src/state/` | Состояние приложения (Store) |
| `src/ui/` | Интерфейс, SVG-холст |
| `.pldf/` | Концепция, дизайн, архитектура, план PLDF |
| `.pldf/ui-prototype/` | HTML-прототип интерфейса |

## Технологии

- Vanilla JavaScript (ES-модули)
- Vite 6 — сборка и dev-сервер
- Vitest — модульные тесты
- SVG — отрисовка графа
